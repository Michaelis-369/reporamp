# RepoRamp ⚡

> Drop a GitHub URL. Understand any codebase in seconds.

Built for the IBM Bob Hackathon — May 2026.
Uses **IBM Bob IDE** for development + **IBM watsonx.ai Granite 3.8b** for analysis.

## What it does

1. **Architecture Overview** — summary, tech stack, key components
2. **Learning Path** — ordered steps telling you exactly which files to read first
3. **Code Chat** — ask Granite anything about the codebase

## Stack

| Layer    | Tech                          |
|----------|-------------------------------|
| AI       | IBM watsonx.ai Granite 3.8b   |
| Backend  | Python Flask                  |
| Frontend | React + Vite                  |
